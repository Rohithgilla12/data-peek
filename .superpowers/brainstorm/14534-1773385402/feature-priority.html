Click to select your top 3 priorities — we'll build all 6, but order matters

1

Data Masking

Blur or hide sensitive columns in query results. Great for demos, screenshots, and pair programming. Toggle per-column or via rules (e.g. mask all "email" columns).

2

Column Statistics

One-click data profiling — show min, max, null count, distinct count, avg, and value distribution per column. Appears as a slide-out panel or inline below the header.

3

Data Generator

Generate realistic fake data for testing. Schema-aware — reads your table structure and generates matching data with Faker. Supports foreign key relationships.

4

CSV Import

Drag-and-drop or file-pick a CSV to import into an existing or new table. Preview data, map columns, handle type conversion, and show progress.

5

Database Notifications (PostgreSQL)

Listen to NOTIFY channels in real-time. See events streaming in as they happen. Subscribe/unsubscribe to channels, with a live event log.

6

Connection Health Monitor

Dashboard showing connection pool stats, active queries, database size, uptime, and replication status. Auto-refresh with configurable intervals.
