Data Masking — How should masked values look?

When a column is masked, what do users see in place of the real data?

A

Blur Effect

CSS blur filter on the cell text. Data is still there but unreadable. Hover to peek (optional). Looks slick in screenshots.

id email 🔒 name
1 john@example.com John Doe
2 jane@company.io Jane Smith
3 bob@startup.dev Bob Wilson
B

Redacted (••••••)

Replace values with dots or asterisks. Clean, unambiguous. No data leaks even if someone inspects the DOM.

id email 🔒 name
1 •••••••••••• John Doe
2 •••••••••••• Jane Smith
3 •••••••••••• Bob Wilson
C

Partial Mask

Show first/last few characters with the middle masked. Useful for identifying records while hiding sensitive parts. e.g. jo••@••••le.com
