Data Generator — Where does it live?

How should users access the fake data generator?

A

Standalone Dialog

Accessible from context menu (right-click table → "Generate Data") or command palette. Opens a dialog where you configure row count and per-column generators.

B

Dedicated Tab

Opens as a new tab (like Table Designer). More room for configuration — preview generated rows, see SQL, configure FK relationships between tables. Better for bulk generation across multiple tables.

C

Inline in Add Row Sheet

Extend the existing "Add Row" sheet with a "Generate" toggle. When enabled, each field shows a generator dropdown instead of manual input. Keeps it lightweight and discoverable.
