## Feature Overview

| Feature                    | Description                                        |
| -------------------------- | -------------------------------------------------- |
| **AI Assistant**           | Natural language to SQL, chart generation (BYOK)   |
| **Multi-tab interface**    | Work on multiple queries simultaneously            |
| **Multi-window support**   | Open multiple database windows                     |
| **Command palette**        | Quick access to all actions with `Cmd+K`           |
| **Query history**          | Every query is saved automatically                 |
| **Watch Mode**             | Re-run a SELECT on a cadence with live diffs       |
| **Time Machine**           | Snapshot, scrub, and diff past query results       |
| **Cross-tab references**   | Reference another tab's result with `@name`        |
| **Transactions**           | Manual commit/rollback on PostgreSQL               |
| **Triggers**               | View, alter, enable/disable, and drop triggers     |
| **Query telemetry**        | Detailed timing breakdown for performance analysis |
| **Benchmark mode**         | Run queries multiple times for P90/P95/P99 stats   |
| **Cancel queries**         | Stop long-running queries mid-execution            |
| **Foreign key navigation** | Click FK values to jump to related records         |
| **JSON viewer**            | Expandable tree view for JSON columns              |
| **Views & routines**       | Browse views, stored procedures, and functions     |
| **SSH tunnels**            | Connect securely through SSH jump hosts            |
| **Dark/Light themes**      | Choose your preferred appearance                   |
| **Keyboard-first**         | Full keyboard navigation support                   |
| **Auto-updates**           | Automatic updates with toast notifications         |
